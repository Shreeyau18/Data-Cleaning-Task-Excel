# Data-Cleaning-Task-Excel
Excel Data Cleaning and Preprocessing Task
## Tool Used
Microsoft Excel
## Dataset
Customer Demographics and Sales Data
## Cleaning Steps Performed
- Identified and handled missing values using filters and blanks selection
- Removed duplicate records using Remove Duplicates tool
- Standardized categorical text values (lowercase, trimmed spaces)
- Renamed column headers to lowercase with underscores
- Converted date columns to consistent dd-mm-yyyy format
- Corrected numeric data types (age, order value, product indicators)
- Removed unnecessary/sensitive columns
## Outcome
The dataset is now clean, consistent, and ready for analysis or modeling.
